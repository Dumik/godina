Вам может быть интересно ознакомиться с некоторыми открытыми API, предоставляющими исторические фотографии:

Flickr Commons

Flickr Commons содержит большую коллекцию исторических фотографий, которые размещены различными библиотеками и музеями.
Сайт: Flickr Commons
API: Flickr API
Europeana

Europeana - это платформа, на которой размещены миллионы культурных и исторических артефактов, включая фотографии, из различных европейских музеев и библиотек.
Сайт: Europeana Collections
API: Europeana API
Library of Congress

Библиотека Конгресса США также предоставляет доступ к историческим фотографиям через свой API.
Сайт: Library of Congress Prints & Photographs
API: Library of Congress Data API
Unsplash

Несмотря на то что Unsplash больше фокусируется на современных фотографиях, здесь также можно найти некоторые исторические фотографии, доступные для бесплатного использования.
Сайт: Unsplash
API: Unsplash API
The New York Public Library Digital Collections

Библиотека предоставляет доступ к миллионам предметов из своих коллекций, включая исторические фотографии.
Сайт: NYPL Digital Collections
API: NYPL Digital Collections API
Пожалуйста, убедитесь, что вы ознакомились и соблюдаете условия использования и лицензионное соглашение каждого API.
